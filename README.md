Descripción
Este proyecto es una aplicación web diseñada para filtrar ventas por un rango de fechas y calcular las comisiones de los vendedores. El sistema consulta una base de datos local y aplica diferentes reglas de porcentaje dependiendo del monto de cada venta realizada. La estructura del código sigue el patrón de arquitectura MVC.

Cómo ejecutar el proyecto
Para poner a funcionar la aplicación en tu computadora, sigue estos pasos:

Instala las dependencias necesarias con el comando: npm install
Inicializa la base de datos con los datos de prueba ejecutando: npm run init-db
Inicia el servidor con el comando: npm start
